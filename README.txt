How to Run:

Method 1:
1. Run:
	$ java -jar Chess.jar

Method 2:
1. Place contents of repository into a directory named JavaChess/
2. From inside JavaChess/ run:
	$ javac -cp .. Chess.java
3. Run:
	$ java -cp .. JavaChess.Chess

Method 3:
1. Place contents of repository into a directory named JavaChess/
2. From the directory directly above JavaChess/ run:
	$ javac JavaChess/Chess.java
3. Run:
	$ java JavaChess.Chess
